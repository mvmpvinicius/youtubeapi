## Installation

1. Git clone the project
2. Run "composer install" and "npm install && npm run dev"
3. Run "php artisan key:generate" in the console
4. Copy .env-example to a new file called .env and set YOUTUBE_API_KEY with your own YouTube API key
5. Run the project "php artisan serve" in the root directory
